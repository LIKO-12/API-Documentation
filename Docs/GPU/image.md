Creats a new image that can be used for drawing.

---

## Usage:

---

#### Syntax:
```lua
img = image(data)
```

---

#### Arguments:

* **data (String)**: The image data, must be in [LK12 GPUImage format](../Files Format/GPU Image.md).

---

#### Returns:

* **img (GPUImage)**: The created image object.
